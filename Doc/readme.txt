【*】程序简介

-工程名称：Tempalte
-工程名称：Debug(将开发板 BOOT0 BOOT1 接高电平，在内部SRAM中运行)
-实验平台: 野火STM32 F429 V2 开发板 
-实验代码；基于野火 触控画板 程序更改

【 ！】功能简介：

触控屏幕控制舵机，并实时显示角度及占空比


【 ！！】注意事项：

字库文件在外部 SPI Flash （Standard_Address 1360*4096 | vertical_Address 3812*4096）
乱码需切换成英文显示函数


【 ！】实验操作：

舵机连线
棕色——负极
红色——正极
橙色——信号线（PA5）

下载程序，即可运行。

【*】 时钟

A.晶振：
-外部高速晶振：25MHz
-RTC晶振：32.768KHz

B.各总线运行时钟：
-系统时钟 = SYCCLK = AHB1 = 180MHz
-APB2 = 90MHz 
-APB1 = 45MHz

  不使用
